Visual AWR -  Readme

Author: Stephen Furlong, steve.furlong@oracle.com

Visual AWR is not supported or endorsed by Oracle.  It was developed independenly by me and is inteded for experienced DBAs and performance engineers.

This project is made possible by the seminal database performance work done by Tyler Muth from Oracle. Specifically his work with the AWRMiner project was leveraged as a guide for the SQL queries needed to extract data from the Oracle AWR tables.

Diagnostic Pack License Required

This script queries DBA_HIST_ views in the Automatic Workload Repository (AWR). To generate AWR reports, take AWR snapshots, or queries these views requires a license for the Diagnostic Pack (12.1 document, 11.2 document or 10.2 document). You should NOT run this script on any Database that is not licensed for the Diagnostic Pack .

 

This tool will chart AWR performance metrics by directly attaching to the database or by reading the output file from an AWRMiner collection script.

Quick Start

AWR performance metrics can be visualized from one of two data sources: JDBC Connection or from an AWRMiner text file.

Collecting AWR metrics from a JDBC Connection

1. Connect to the target database using the DB Connections screen below and save the connection.

2. Use the Chart DB Data wizard to collect the AWR snapshot data and chart the metrics.
