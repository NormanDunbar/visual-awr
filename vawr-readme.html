Visual AWR Readme - v02h

Author: Stephen Furlong, steve.furlong@oracle.com

Project Overview: The Visual AWR toolkit provides fine grained graphical trend analysis for Oracle Database AWR (Automatic Workload Repository) and OSWatcher/ExaWatcher performance metrics. AWR & OSWatcher/ExaWatcher metrics are collected automatically by Oracle systems. This raw data can be difficult to interpret unless aggregated over multiple days and charted for trend and root cause analysis. VisualAWR is intended for use by experienced DBAs and performance engineers to diagnose workload performance issues and inform platform capacity planning.

Acknowledgement: This project owes a debt to the seminal database performance work done by Tyler Muth from Oracle. Specifically his work with the AWRMiner project was leveraged as a guide for the SQL queries used to extract data from the Oracle AWR tables.

Quick Start

See my video guide on YouTube and my blog here.

This tool will chart AWR performance metrics by directly attaching to the database or by reading the output file from an AWRMiner collection script. AWR performance metrics can be visualized from one of two data sources: JDBC Connection or from an AWRMiner text file.

Charting AWR metrics from a JDBC Connection

1. Connect to the target database using the DB Connections screen below and save the connection.

2. Use the Chart DB Data wizard to collect the AWR snapshot data and chart the metrics.

3. Follow the wizard and chart the specific metrics you are interested in.

Charting AWR metrics from an AWRMiner collection file.

1. Choose a valid AWRMiner output file and select a metric to chart.
